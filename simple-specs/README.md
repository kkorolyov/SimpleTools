# SimpleSpecs
A collection of Groovy utilities for simplifying tests and specifications.

## Components
### Reflective Field Manipulation
One-line getters/setters for static and instance fields

### Random Generators
* `int`
* `byte`
* `byte` array
* `short`
* `long`
* `float`
* `double`
* `String`
